## Maven
To use the RedstoneAPI in your projects you can use Maven to implement it.
Implement the Github repository by pasting this in your pom.xml

Replace {VERSION} with the current version.
```xml
<repositories>
    <repository>
        <id>github</id>
        <name>GitHub Redstonecrafter0 Apache Maven Packages</name>
        <url>https://maven.pkg.github.com/Redstonecrafter0/RedstoneAPI</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
        <groupId>net.redstonecraft</groupId>
        <artifactId>redstoneapi</artifactId>
        <version>{VERSION}</version>
    </dependency>
</dependencies>
   ```
